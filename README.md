# Simple Interest Calculator

This project is a simple interest calculator that helps users calculate the interest earned or paid on a principal amount over a period of time at a given interest rate.

## 📌 Formula Used

Simple Interest is calculated using the formula:

SI = (P × R × T) / 100

Where:
- **P** = Principal amount
- **R** = Rate of interest (per year)
- **T** = Time (in years)

## 🚀 Features

- Easy to use
- Fast calculation
- Beginner-friendly project
- Helps understand basic financial concepts

## 🛠️ Technologies Used

- HTML (for structure)
- CSS (for styling)
- JavaScript (for logic)

## 📖 How to Use

1. Enter the **Principal amount**
2. Enter the **Rate of Interest**
3. Enter the **Time period**
4. Click on **Calculate**
5. The simple interest will be displayed

## 💡 Example

If:
- Principal = 1000  
- Rate = 5%  
- Time = 2 years  

Then:

SI = (1000 × 5 × 2) / 100 = 100
